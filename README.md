# bar_finding
This is a repo of a pipeline that extract bar information from galaxy images described in Deng+2023.
Dependencies:\n
SEP (https://sep.readthedocs.io/en/stable/index.html)\n
SCARLET (https://pmelchior.github.io/scarlet/)\n
photutils (https://photutils.readthedocs.io/en/stable/index.html)\n
savitzky_golay_werrors (https://github.com/surhudm/savitzky_golay_with_errors)\n
numpy, pandas, astropy, scipy
